- Start COM port emulator (suggested emulator: Virtual Serial Port Driver Pro).

- Run Serial Data Transmitter and Serial Data Receiver simultaneously (utilize Multiple Startup Projects setting).

- Press "Receive" on the Serial Data Receiver window.

- Then press "Transmit" on the Serial Data Receiver window.

- A natural language report will be displayed line-by-line in the console output and then written to a text file located @"serial_comms_exercise\SerialDataExercise\SerialDataReceiver\bin\Report.txt"
